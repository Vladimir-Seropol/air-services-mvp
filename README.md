# MVP сервиса дополнительных услуг для авиапассажиров

Небольшое тестовое задание для вакансии Frontend-разработчика.

## Стек
- React
- TypeScript
- Vite

## Функциональность
- отображение списка дополнительных услуг
- добавление услуг в заказ
- подсчёт общей суммы
- адаптивная вёрстка (mobile-first)

## Запуск проекта
npm install
npm run dev
